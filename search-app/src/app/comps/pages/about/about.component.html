This is a technical test for Managed 24/7


Senior Javascript Technical Interview

Deliverable:

Please create a repository/repositories on github and send us the link so we can checkout your work. You may use the readme file to explain your technical choices and eventually let us know what else you would have done if you had more time.

Instructions:

The goal of this exercise is to gauge your proficiency with the MEAN stack. You are free to spend as much time as you wish on the exercise, please just make sure you commit your changes as regularly as possible so we can judge fairly your work compared to other candidates.

Part 1 - Node / Express / MongoDB:

Create a node.js restful API with a single endpoint that given a search term returns a list of names that begin with that term. The query will be done against MongoDB. The endpoint must be secured.

Part 2 - Angular

Create an angular 2+ application that uses the node.js application you just created to list names. The application will also have a search box that will filter out the list based on the search.
